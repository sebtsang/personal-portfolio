# Sebastian Tsang Portfolio

Single-page Astro portfolio built for a clean, editorial-minimal presentation of experience, current AI focus, and public writing.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Push the repository to GitHub.
2. Import the project in Vercel.
3. Keep the default Astro build settings:
   - Build command: `npm run build`
   - Output directory: `dist`
